# readme generator

node profile generator

## Getting Started

run "node index.js" in terminal

### Prerequisites

 
```
   install node
```

### Installing

install the following npm packages using the "npm i" command; 

  axios
  electron 
  electron-html-to 
  fs
  inquirer
